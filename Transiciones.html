TUTORIAL CSS3 - Transiciones

Transición del color de fondo

Este código lo que hará es, que cuando estemos encima del texto que queremos, haya una transición y se cambie el color de fondo.

La etiqueta background-color la usaremos para cambiar el color de fondo.

La etiqueta -webkit-transition es la que nos permite hacer la transición, y dentro de ella, le indicamos que queremos que se haga la transición del color de fondo, y que dure 1 segundo.

.caja1{
background-color: #C0392B;
-webkit-transition: background-color 1s linear;
transition: background-color 1s linear;
}
.caja1:hover{
background-color: #3F51B5;
}

Transición de sombra

Este código lo que hará es, que cuando estemos encima del texto que queremos, haya una transición mostrando la sombra

La etiqueta box-shadow la usaremos para cambiar la sombra.

.caja6{
  -webkit-transition: 1s linear;
  transition: 1s linear;
  background-color: #F1C40F;
}
.caja6:hover{
  -webkit-box-shadow: 10px 10px 14px 2px rgba(0,0,0,0.47);
  box-shadow: 10px 10px 14px 2px rgba(0,0,0,0.47);
}
