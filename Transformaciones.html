TUTORIAL CSS3 - Transformaciones

Propiedades para modificar las transformaciones de los elementos de un documento HTML

transform-matrix

La función CSS matrix() define una matriz de transformación 2D homogénea. Su resultado es un tipo de datos <transformación-función>.

Sintaxis

transform: matrix(1.2, 0.2, -1, 0.9, 0, 20);
transform: matrix(a, b, c, d, tx, ty);

valores admitidos

  • a b c d
    Son <números> que describen la transformación lineal.
  • tx ty
    Son <números> que describen la traslación a aplicar.

Los primeros cuatro valores hacen referencia a la transformación que sufre el objeto que estamos modificando

Los ultimos dos valores hacen referenciaa la traslación que sufre el objeto que estamos modificando

La diferencia esta en que los primeros cuuatro valores modifican su tamaño y los ultimos dos valores modifican su posición respecto al eje de coordenadas original

Ejemplos:


En el siguiente ejemplo se puede ver como se modifica el tamaño y la posición de un elemento

.cuadrado {
  width: 100px;
  height: 100px;
  background-color: #ff0000;
  transform: matrix(1.2, 0.2, -1, 0.9, 0, 20);
}

transformaciones-perspective

La función CSS perspective() define una transformación que establece la distancia entre el usuario y el plano z=0, la perspectiva desde la que estaría el espectador si la interfaz bidimensional fuera tridimensional.

Su resultado es un tipo de datos <transformación-función>.

Sintaxis

perspective(d);

valores admitidos

  • d
    Es una <longitud> que representa la distancia del usuario al plano z=0. Si es 0 o un valor negativo, no se aplica ninguna transformación de perspectiva.

Ejemplos:

style=transform: perspective(800px);

transformaciones-rotate

La propiedad CSS rotate permite especificar las transformaciones de rotación de forma individual e independiente de la propiedad transform.

Esto se ajusta mejor al uso típico de la interfaz de usuario, y ahorra tener que recordar el orden exacto de las funciones de transformación para especificar en la propiedad transform.

Sintaxis

/* Keyword values */
rotate: none;

/* Angle value */
rotate: 90deg;
rotate: 0.25turn;
rotate: 1.57rad;

/* x, y, or z axis name plus angle */
rotate: x 90deg;
rotate: y 0.25turn;
rotate: z 1.57rad;

/* Vector plus angle value */
rotate: 1 1 1 90deg;

/* Global values */
rotate: inherit;
rotate: initial;
rotate: revert;
rotate: revert-layer;
rotate: unset;
  • Valor del ángulo
    Un ángulo(<angle>) que especifica el ángulo de rotación del elemento afectado, alrededor del eje Z. Equivale a la función rotate() (rotación 2D).

  • Nombre del eje x, y o z más el valor del ángulo
    El nombre del eje alrededor del cual se quiere girar el elemento afectado ("x", "y" o "z"), más un ángulo(<angle>) que especifica el ángulo por el que se girará el elemento. Equivale a una función rotateX()/rotateY()/rotateZ() (rotación 3D).

  • Vector más valor angular
    Tres <number>s que representan un vector centrado en el origen que define una línea alrededor de la cual se quiere girar el elemento, más un ángulo(<angle>) que especifica el ángulo por el que se girará el elemento. Equivale a una función rotate3d() (rotación 3D).

  • none
    Especifica que no se debe aplicar ninguna rotación.

translateX(x)

La función CSS translateX() translada un elemento horizontalmente en el plano 2D.

Sintaxis

transform: translateX(size);

Admite un valor

  • El valor se aplica al desplazamiento en el eje x.

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

transform: translateX(0);
transform: translateX(400px);
transform: translateX(-2rem);
transform: translateX(30ch);

translateY(y)

La función CSS translateY() translada un elemento verticalmente en el plano 2D.

Sintaxis

transform: translateY(size);

Admite un valor

  • El valor se aplica al desplazamiento en el eje y.

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

transform: translateY(0);
transform: translateY(45px);
transform: translateY(-3rem);
transform: translateY(3ch);

translate()

La function de CSS translate() recoloca un elemento en el eje horizontal y/o vertical.

Sintaxis

transform: translate(size , [size]);

Admite uno o dos valores

  • El valor se aplica al desplazamiento en el eje x o/y y.

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

transform: translate(20px);
transform: translate(50px,40px);
transform: translate(-3ch, -3mm);
transform: translate(2.1rem, -2ex);

translateZ()

La función translateZ() CSS reposiciona un elemento a lo largo del eje-z (z-axis) en el espacio 3D, es decir, más cerca o mas lejos del espectador.

Sintaxis

transform: perspective translateZ(size);

Admite un valor

  • El valor se aplica al desplazamiento en el eje z.

La función perspective() posiciona al espectador en relación con el plano que se encuentra donde z=0

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

transform: perspective(100px) translateZ(60px);
perspective(100px) translateZ(-15rem);
transform: perspective(100px) translateZ(4ch);

translate3d()

La función CSS translate3d() recoloca un elemento en el espacio 3D.

Sintaxis

transform: translate(size , [size]);

Admite de uno a tres valores

  • El valor se aplica al desplazamiento en el eje x,y o/y z.

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

transform: perspective(200px) translate3d(42px, -62px, 70px);
transform: perspective(200px) translate3d(-4rem, 0.5rem, -8rem);
transform:  perspective(200px) translate3d(3ch, 1ch, 7ch);

Sintaxis

translate(sizeX , [sizeY], [sizeZ]);

Admite de uno a tres valores

  • El valor se aplica al desplazamiento en el eje x, y o/y z.

Si se indica un valor negativo, el elemento se desplazará en la dirección opuesta.

Ejemplos:

scaleX(x)

La función CSS scaleX() define una transformación que cambia el tamaño de un elemento a lo largo del eje x (horizontalmente).

Sintaxis

transform: scaleX(size);

Admite un valor

  • Reescala el elemento un número de veces (sólo en horizontal).
  • Solo admite porcentajes y números decimales

Si se indica un valor negativo, el elemento se escala de forma inversa.

Ejemplos:

transform: scaleX(2);
transform: scaleX(50%);
transform: scaleX(-3.5);
transform: scaleX(-20%);

scaleY(y)

Reescala el elemento un número de veces (sólo en vertical).

Sintaxis

transform: translateY(size);

Admite un valor

  • Reescala el elemento un número de veces (sólo en vertical).
  • Solo admite porcentajes y números decimales

Si se indica un valor negativo, el elemento se escala de forma inversa.

Ejemplos:

transform:scaleY(1.5);
transform: scaleY(300%);
transform: scaleY(-2.5);
transform: scaleY(-20%);

scale()

La función CSS scale() define una transformación que modifica el tamaño de un elemento en el plano 2D. Debido a que la cantidad de escalado está definida por un vector, puede cambiar el tamaño de la dimensiones horizontal y vertical a diferentes escalas.

Sintaxis

transform: translate(size , [size]);

Admite uno o dos valores

  • Reescala el elemento un número de veces (sólo en vertical y/o horizontal).
  • Solo admite porcentajes y números decimales

Si se indica un valor negativo, el elemento se escala de forma inversa.

Ejemplos:

transform: scale(2,3);
transform: scale(140%,50%);
transform: scale(-4, -3);
transform: scale(-40%, -300%);

scaleZ()

La función CSS scaleZ() modifica la coordenada z de cada punto del elemento por un factor constante

Sintaxis

transform: perspective translateZ(size);

Admite un valor

  • Moodifica la coordenada z.
  • Si el escalado es 1 no se modifica
  • Solo admite porcentajes y números decimales

La función perspective() posiciona al espectador en relación con el plano que se encuentra donde z=0

Si se indica un valor negativo, el elemento se escala de forma inversa.

Ejemplos:

transform: perspective(300px) scale3d(3,4,5) rotateX(45deg);
transform: perspective(300px) scale3d(10%,40%,50%) rotateX(45deg);
transform: perspective(300px) scale3d(-1,150%,2) rotateX(45deg);

scale3d()

La función CSS scale3d() define una transformación que cambia el tamaño de un elemento en el espacio 3D. Debido a que la cantidad de escala está definida por un vector, puede cambiar el tamaño de diferentes dimensiones en diferentes escalas.

Sintaxis

transform: translate(size , [size]);

Admite de uno a tres valores

  • El valor se aplica al escalado en el eje x,y o/y z.

Si se indica un valor negativo, el elemento se escala de forma inversa.

Ejemplos:

transform: perspective(300px) scale3d(2,1,5) rotateX(45deg);
transform: perspective(300px) scale3d(10%,40%,50%) rotateX(45deg);
transform:  perspective(300px) scale3d(-1,150%,2) rotateX(45deg);

transformaciones-skew

La función CSS skew() define una transformación que sesga un elemento en el plano 2D. Su resultado es un tipo de datos <transform-function>.

Esta transformación es un mapeo de cizallamiento (transvección) que distorsiona cada punto dentro de un elemento en un determinado ángulo en las direcciones horizontal y vertical. El efecto es como si se cogiera cada esquina del elemento y se tirara de ellas en un ángulo determinado.

Las coordenadas de cada punto se modifican en un valor proporcional al ángulo especificado y a la distancia al origen. Así, cuanto más lejos del origen esté un punto, mayor será el valor que se le añada.

Sintaxis

La función skew() se especifica con uno o dos valores, que representan la cantidad de inclinación que se aplicará en cada dirección. Si sólo se especifica un valor se utiliza para el eje x y no habrá inclinación en el eje y. 

skew(ax)

skew(ax, ay)
  • ax
    Es un <ángulo> que representa el ángulo a utilizar para distorsionar el elemento a lo largo del eje x (o abscisa).

  • ay
    Es un <ángulo> que representa el ángulo a utilizar para distorsionar el elemento a lo largo del eje y (u ordenada). Si no se define, su valor por defecto es 0, lo que resulta en una inclinación puramente horizontal.

Ejemplos:

Inclinación en el eje x solamente

transform: skew(10deg);
Normal
Skewed

Inclinación en ambos ejes

 transform: skew(10deg, 10deg);
Normal
Skewed
