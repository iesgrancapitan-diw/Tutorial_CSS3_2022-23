TUTORIAL CSS3 - Pseudos Css3

Un PseudoElemento CSS es una palabra clave que se añade a los selectores y que especifica un elemento especial del elemento seleccionado.
Una PseudoClase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.

PseudoElementos

Ir a Pseudo Clases

::after

Selecciona el elemento al que se puede aplicar estilo que aparece a continuación del contenido del elemento que lo origina.

Sintaxis

elemento::after {CSS a aplicar}

Admite todas las propiedades css y ademas la propiedad "content".

La propiedad "content" es obligatoria y se utiliza para especificar el contenido del pseudoElemento.

El valor de la propiedad "content" puede ser:

  • cadena: Define el contenido de texto. El valor por defecto es "".
  • url: Define una dirección URL
  • counter (name)
  • counter (name, list-style-type)
  • counter (name, list-style-type, separator)
  • counters (name, string)
  • counters (name, string, list-style-type)
  • counters (name, string, list-style-type, separator)
  • attr(name): Define un atributo de un selector que será mostrado antes/después del selector.
  • open-quote
  • close-quote
  • no-open-quote
  • no-close-quote

Es comunmente usado para añadir contenido cosmético a un elemento con la propiedad content. Es en linea (inline) de forma predeterminada.

Ejemplos:

#ejemplo::after {
    content: "→";
    color: #f00;
}

Resultado:

A este texto se le añade una flecha roja

::before

Crea un pseudoelemento que es el primer hijo del elemento seleccionado. Es usado normalmente para añadir contenido estético a un elemento, usando la propiedad content. Este elemento se muestra en línea con el texto de forma predeterminada.

Sintaxis

elemento::before{CSS a aplicar}

Admite todas las propiedades css y ademas la propiedad "content".

La propiedad "content" es obligatoria y se utiliza para especificar el contenido del pseudoElemento.

El valor de la propiedad "content" puede ser:

  • cadena: Define el contenido de texto. El valor por defecto es "".
  • url: Define una dirección URL
  • counter (name)
  • counter (name, list-style-type)
  • counter (name, list-style-type, separator)
  • counters (name, string)
  • counters (name, string, list-style-type)
  • counters (name, string, list-style-type, separator)
  • attr(name): Define un atributo de un selector que será mostrado antes/después del selector.
  • open-quote
  • close-quote
  • no-open-quote
  • no-close-quote

Es comunmente usado para añadir contenido cosmético a un elemento con la propiedad content. Es en linea (inline) de forma predeterminada.

Ejemplos:

    content: "♥";

Resultado:

A este texto se le añade un corazón al inicio.

::first-letter

Aplica estilos a la primera letra de la primera línea un elemento de bloque, sólo cuando no es precedido de otro contenido (como imágenes o tablas).

Sintaxis

elemento::first-letter {CSS a aplicar}

Propiedades admitidas:

  • Todas las propiedades de fuente
  • Todas las propiedades de fondo
  • Todas las propiedades de márgenes
  • Todas las propiedades de relleno
  • Todas las propiedades de bordes
  • La propiedad color
  • Las propiedades text-decoration, text-shadow, text-transform, letter-spacing, word-spacing, line-height, text-decoration-line, text-decoration-style, box-shadow, float y vertical-align

Ejemplos:

#ejemplo3::first-letter {
    color: #f00;
    font-size: 2em;
}

Resultado:

La primera letra de este parrafo es roja y más grande.

Propiedades admitidas:

  • Todas las propiedades de fuente
  • Todas las propiedades de fondo
  • Todas las propiedades de márgenes

  • Las propiedades text-decoration, text-shadow, text-transform, letter-spacing, word-spacing, line-height, text-decoration-line, text-decoration-style, box-shadow, float y vertical-align

Ejemplos:

    color: blue;
    background-color: #f00;

::marker

Selecciona la caja de marcadores de un elemento de la lista, que normalmente contiene una viñeta o un número

Sintaxis

ul li::marker {CSS a aplicar}

Propiedades admitidas:

  • Todas las propiedades de fuente
  • La propiedad text-combine-upright, unicode-bidi y direction
  • La propiedad content
  • La propiedad color

Ejemplos:

.ejemplo5::marker {
    color: blue;
    font-size: 2em;
}

Resultado:

  • Marca con color azul
  • Marca con color azul 2
  • Marca con color azul 3

:active

Representa un elemento (como un botón) que el usuario está activando.

Sintaxis

elemento:active {CSS a aplicar}

Admite todas las propiedades css.


La pseudo-clase :active se usa comúnmente en los elementos "a" y "button", pero también se puede usar en otros elementos.

Ejemplos:

#ejemploPClase:active {
    color: #f00;
    background-color: #0f0;
}

Resultado:

El estilo se aplica al hacer click en este elemento
