TUTORIAL CSS3 - Efectos de Texto y Fuentes

Propiedades para modificar el texto y la fuente de un documento HTML

text-align

Se usa para establecer la alineación horizontal del texto

Sintaxis

text-align: left|right|center|justify|initial|inherit;

Admite cuatro valores, que son los siguientes:

  • left: alinea el texto a la izquierda.
  • right: alinea el texto a la derecha.
  • center: centra el texto
  • justify: justifica el contenido, es decir, iguala la longitud de todas las líneas del texto

Valor por defecto

left si el texto se escribe de izquierda a derecha (valor ltr en la propiedad direction) y right si el texto se escribe de derecha a izquierda (valor rtl en la propiedad direction)

Ejemplos:

text-align: center;
text-align: left;
text-align: right;
text-align: justify;
