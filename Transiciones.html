TUTORIAL CSS3 - Transiciones

Las transiciones te permiten cambiar el valor propiedades de forma fluida durante un tiempo determinado.

transition-duration

Especifica cuántos segundos o milisegundos tarda en completarse un efecto de transición.

Sintaxis

transition-duration: [duración de la transición]s;

Admite de un valor:

  • time: indica el tiempo en segundos o milisegundos que llevara hasta completar la transicion.

Ejemplos:

.transiciones_transition-duration {
    width: 6em;
    height: 3em;
    background-color: #0000FF;
    color: white;
    font-weight: 300;
    transition-duration: 3s;
}

.transiciones_transition-duration:hover {
    background-color: #f8f403;
    color: black;
    width: 200px;
    height: 200px;
}

transition-delay

La propiedad de transition-delay especifica un retraso (en segundos) para el efecto de transición.

Sintaxis

transition-delay: [el retardo en segundos]s;

Ejemplos:

.transiciones_transition-delay {
    width: 100px;
    height: 100px;
    background: red;
    transition-duration: 3s;
    transition-delay: 1s;
}
                
.transiciones_transition-delay:hover {
    width: 300px;
}

transition-property

Especifica el nombre o nombres de las propiedades CSS a las que se deben aplicar las transiciones. Sólo las propiedades listadas aquí son animadas durante las transiciones; los cambios en todas las demás propiedades ocurren instantáneamente como es habitual.

Sintaxis

transition-property: none|all|property|initial|inherit;

Valores:

  • all: Es el valor por defecto e indica que se verán afectadas por la transición todas las propiedades que se puedan.
  • none: Indica que no se aplicará la transición a ninguna propiedad
  • property: Nombre de la propiedad que se va a ver afectada; si son varias, separarlas con comas.
  • initial: Se utiliza para establecer una propiedad CSS a su valor predeterminado.
  • inherit: Se heredan las características del elemento padre

Ejemplos:

.transiciones_transition-property {
    font-size: 14px;
    width: 75px;
    height: 75px;
    background: rgb(255, 174, 1);
    transition-property: width, height, font-size;
    transition-duration: 2s;
}
                
.transiciones_transition-property:hover {
    font-size: 25px;
    width: 250px;
    height: 120px;
}

transition-timing-function

Especifica la curva de velocidad del efecto de transición.

Esta propiedad permite que un efecto de transición cambie de velocidad durante su duración.

Sintaxis

transition-timing-function: linear|ease|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end|jumpstep)|cubic-bezier(n,n,n,n)|initial|inherit;

Los posibles valores son:

  • linear: Especifica un efecto de transición con la misma velocidad de principio a fin (equivalente a cubic-bezier (0,0,1,1))
  • ease: Valor por defecto. Especifica un efecto de transición con un inicio lento, luego rápido y luego un final lento (equivalente a cubic-bezier (0.25,0.1,0.25,1))
  • ease-in: Especifica un efecto de transición con un inicio lento (equivalente a cubic-bezier (0,0,0.58,1))
  • ease-out: Especifica un efecto de transición con un final lento (equivalente a cubic-bezier (0,0,0.58,1))
  • ease-in-out: Especifica un efecto de transición con un inicio y un final lentos (equivalente a cubic-bezier (0.42,0,0.58,1))
  • step-start: Equivalente a steps(1, start).
  • step-end: Equivalente a steps(1, end).
  • steps(int,start|end|jumpterm): Especifica una función paso a paso, con dos parámetros. El primer parámetro especifica el número de intervalos en la función. Debe ser un número entero positivo. El segundo parámetro, que es opcional, es el valor "start" o "end", y especifica el punto en el que se produce el cambio de valores dentro del intervalo. Si se omite el segundo parámetro, se le asigna el valor "end".
    • jump-start la función continua a la izquierda, de modo que el primer salto ocurre cuando comienza la transición.
    • jump-end la función continua a la derecha, de modo que el primer salto ocurre cuando termina la transición.
    • jump-noneNo hay salto en ninguno de los extremos. En su lugar, se mantiene en la marca del 0% y en la marca del 100%, cada uno durante 1/n de la duración
    • jump-bothIncluye pausas en las marcas 0% y 100%, agregando un paso durante el tiempo de transición.
  • cubic-bezier(n,n,n,n):Puedes definir tus propios valores en la función cubic-bezier. Los valores posibles son valores numéricos de 0 a 1.

Ejemplos:

.transiciones_transition-timing-function_ease {
    transition-timing-function: ease;
}
                
.transiciones_transition-timing-function_easein {
    transition-timing-function: ease-in;
}
                
.transiciones_transition-timing-function_easeout {
    transition-timing-function: ease-out;
}
                
.transiciones_transition-timing-function_easeinout {
    transition-timing-function: ease-in-out;
}
                
.transiciones_transition-timing-function_linear {
    transition-timing-function: linear;
}
                
.transiciones_transition-timing-function_cb {
     transition-timing-function: cubic-bezier(0.2, -2, 0.8, 2);
}

transition

La propiedad transition es una propiedad abreviada de transition-property, transition-duration, transition-timing-function (en-US), y transition-delay. Permite definir la transición entre dos estados de un elemento. Hay diferentes estados que pueden ser definidos utilizando pseudo-clases como :hover o :active o aplicado dinámicamente usando JavaScript.

Sintaxis

transition: property duration timing-function delay;

Admite de uno a cuatro valores, que son los siguientes:

  • transition-property: especifica el nombre de la propiedad CSS sobre la que se hará la transicion.
  • transition-duration: especifica el tiempo el segundos o milisegundos que durará la transicion
  • transition-timing-function: da la velocidad de la curva de la transición
  • transition-delay: define cuando empezará la transición

Para crear una transición con la propiedad transition y usando el shorthand se debe indicar la propiedad del elemento a la que se desea aplicar la transición seguida de las propiedades de transiciones que se le desea aplicar (tiempo, retardo, curva, etc.) no es necesario poner todas las proiedades de transiciones pero al menos si la propiedad del elemento para saber que se desea cambiar. Estas propiedades del elemento se pueden separar en bloques con el uso de "," entre las propiedades de transiciones solo basta separarlas con espacios.

Ejemplos:

/* Asi sería sin shorthand*/ 
.transiciones_transition {
    border-style: solid;
    border-width: 1px;
    display: block;
    width: 100px;
    height: 100px;
    background-color: #0000FF;
    transition-property: width, height, background-color, transform;
    transition-duration: 2s;
}
                    
/* Así sería con shorthand*/
.transiciones_transition {
    border-style: solid;
    border-width: 1px;
    display: block;
    width: 100px;
    height: 100px;
    background-color: #0000FF;
    transition: width 2s, height 2s, background-color 2s, transform 2s;
}

.transiciones_transition:hover {
    background-color: #c001b6;
    width: 200px;
    height: 200px;
    transform: rotate(180deg);
}
