Propiedades para modificar los fondos de los elementos de un documento HTML

Background-color

Es un propiedad de CSS que define el color de fondo de un elemento, puede ser el valor de un color o la palabra clave transparent.

Sintaxis

background-color: [color | transparent | inherit];

Admite 2 valores, que son los siguientes:

  • Un color: Se puede especificar el color de fondo como un valor RGB(A) hexa-decimal o regular, con valores HSL(A) o utilizando una de las palabras claves predefinidas de color.
  • Transparent: El valor por defecto del color de fondo (background-color) es transparent tomando así el color del elemento que está justo debajo de él en el apilamiento.

background-attachment

Determina si la posición de la imagen de fondo será fija dentro de la pantalla o se desplazará con su bloque contenedor.

Sintaxis

background-attachment: scroll|fixed|local|initial|inherit;

Ejemplos: 

    .fondos-background-attachment-imagen1{
    background-image: url("../img/background-attachment-imagen.png");
     background-attachment: scroll;
}

  • Scroll

    - La imagen de fondo se desplazará con la página. esto es por defecto

    • .fondos-background-attachment-imagen2{
    background-image: url("../img/background-attachment-imagen.png");
    background-attachment: fixed;
}

  • fixed

    - La imagen de fondo no se desplazará con la página.

    • .fondos-background-attachment-imagen3{
    background-image: url("../img/background-attachment-imagen.png");
    background-attachment: local;
}

  • local

    - La imagen de fondo se desplazará con el contenido del elemento.

    • .fondos-background-attachment-imagen4{
    background-image: url("../img/background-attachment-imagen.png");
    background-attachment: initial;
}

  • initial

    - Establece esta propiedad en su valor predeterminado.

  • inherit

    - Hereda esta propiedad de su elemento padre.

Background-image

Establece una o más imágenes de fondo para un elemento.

De forma predeterminada, se coloca una imagen de fondo en la esquina superior izquierda de un elemento y se repite tanto vertical como horizontalmente.

Sintaxis

background-image: url|none|initial|inherit;

Ejemplos: 

    .fondos-background-image-imagen1{
    background-image: url("../img/background-image-imagen.png");
}

  • url

    - La URL de la imagen. Se pueden especificar más imagenes separando con una coma.

    • .fondos-background-image-imagen2{
    background-image: none;
}

  • none

    - No se muestra ninguna imagen por defecto.

    • .fondos-background-image-imagen3{
     background-image: conic-gradient(red, yellow, green);
}

  • conic-gradient

    - Establece un degradado cónico, pero se tienen que definir dos colores como mínimo.

    • .fondos-background-image-imagen4{
    background-image: linear-gradient(red, yellow, rgb(140, 139, 235));
}

  • linear-gradient

    - Establece un degradado lineal, pero se tiene que definir dos colores como mínimo.

    • .fondos-background-image-imagen5{
    background-image: radial-gradient(red, green, rgb(140, 139, 235));
}

  • radial-gradient

    - Establece un degradado radial(del centro a los bordes), teniendo que definir dos colores como mínimo.

    • .fondos-background-image-imagen6{
    background-image: repeating-conic-gradient(red 20%, yellow 25%);
}

  • repeating-conic-gradient

    - Repite un radiante cónico.

    • .fondos-background-image-imagen7{
    background-image: repeating-linear-gradient(red, yellow 10%, green 20%);
}

  • repeating-linear-gradient

    - Repite un radiante lineal.

    • .fondos-background-image-imagen8{
    background-image: repeating-radial-gradient(red, yellow 20%, green 25%);
}

  • repeating-radial-gradient

    - Repite un radiante radial.

  • initial

    - Establece su valor predeterminado.

  • inherit

    - Hereda esta propiedad de su elemento padre.

Background-clip

Define hasta dónde debe extenderse el fondo dentro del elemento.

Sintaxis

background-clip: [border-box|padding-box|content-box];

Explicación del uso de las 3 propiedades:

  • Border-box: El fondo se extiende completamente por todo el elemento, incluso debajo del borde.
  • Padding-box: El fondo solo se extiende hasta el borde del borde: incluye el relleno pero no el borde.
  • Content-box: El fondo solo se extiende hasta el borde del contenido : no incluye el relleno ni el borde.

Ejemplos:

#fondos-clip-border-box{
    background-color: aqua;
    background-clip: border-box;
    border: 5px dotted black;
    margin: 10px;
    padding: 10px;
}

Background-origin

Propiedad especifica la posición de origen de una imagen de fondo.

Sintaxis

background-origin: padding-box|border-box|content-box|initial|inherit;

Ejemplos: 

    .fondos-background-origin-imagen1{
    background: url("../img/background-origin-imagen.png");
    background-origin: padding-box;
}

  • padding-box

    - Valor por defecto. La imagen de fondo comienza desde la esquina superior izquierda del borde de relleno. Se puede poner más de un valor con una coma.

    • .fondos-background-origin-imagen2{
    background: url("../img/background-origin-imagen.png");
    background-origin: border-box;
}

  • border-box

    - La imagen de fondo comienza desde la esquina superior izquierda del borde.

    • .fondos-background-origin-imagen3{
    background: url("../img/background-origin-imagen.png");
    background-origin: content-box;
}

  • content-box

    - La imagen de fondo comienza desde la esquina superior izquierda del contenido.

  • initial

    - Establece esta propiedad en su valor predeterminado.

  • inherit

    - Hereda esta propiedad de su elemento principal.

Background-size

Define el tamaño de la imagen de fondo.

Sintaxis

background-size: [auto | size(px px) | size(% %) | contain | cover];

Definiremos a continuación sus propiedades más detalladas:

  • auto: La imagen de fondo conservará su tamaño original. Por ejemplo, esta imagen de fondo tiene un tamaño de 960 px por 640 px. Su relación de aspecto es de 3 por 2, es más grande que su contenedor (que tiene una altura de 150 px) y, por lo tanto, se recortará .
  • size (pixeles): Puede especificar un tamaño en píxeles: el primer valor es el tamaño horizontal, el segundo es el tamaño vertical.
  • size (porcentaje): Puede utilizar valores porcentuales. Tenga en cuenta que esto puede alterar la relación de aspecto de la imagen de fondo y generar resultados inesperados.
  • contain: La palabra clave contain cambiará el tamaño de la imagen de fondo para asegurarse de que permanezca totalmente visible.
  • cover: La palabra clave cover cambiará el tamaño de la imagen de fondo para asegurarse de que el elemento esté completamente cubierto.

Ejemplos:

#fondos-background-size-auto {
    background-image: url("../img/background-size-img.png");
    background-size: auto;
    height: 150px;
    border: 2px solid;
}


#fondos-background-size-px {
    background-image: url("../img/background-size-img.png");
    background-size: 400px 400px;
    height: 150px;
    border: 2px solid;
}


#fondos-background-size-porcentaje {
    background-image: url("../img/background-size-img.png");
    background-size: 100% 100%;
    height: 150px;
    border: 2px solid;
}


#fondos-background-size-contain {
    background-image: url("../img/background-size-img.png");
    background-size: contain;
    height: 150px;
    border: 2px solid;
}


#fondos-background-size-cover {
    background-image: url("../img/background-size-img.png");
    background-size: cover;
    height: 150px;
    border: 2px solid;
}

Background-repeat

La propiedad CSS establece cómo se repiten las imágenes de fondo.Una imagen de fondo puede repetirse a lo largo de los ejes horizontal y vertical, o no repetirse en absoluto.

Sintaxis

background-repeat: [repeat | repeat-x | repeat-y | no-repeat | space | round ];

Definiremos a continuación más detallado:

  • repeat: La imagen de fondo se repetirá tanto horizontal como verticalmente. La última imagen se recortará si no encaja.
  • repeat-x: La imagen de fondo solo se repetirá horizontalmente.
  • repeat-y: La imagen de fondo solo se repetirá verticalmente.
  • no-repeat: La imagen no se repite (y, por lo tanto, el área de pintura de la imagen de fondo no estará necesariamente cubierta por completo).
  • space: La imagen se repite tanto como sea posible sin recorte. La primera y la última imagen se fijan a ambos lados del elemento y los espacios en blanco se distribuyen uniformemente entre las imágenes.
  • round: A medida que el espacio permitido aumenta de tamaño, las imágenes repetidas se estirarán (sin dejar espacios) hasta que haya espacio (espacio restante >= la mitad del ancho de la imagen) para agregar otra. Cuando se agrega la siguiente imagen, todas las actuales se comprimen para dejar espacio. Ejemplo: una imagen con un ancho original de 260 px, repetida tres veces, puede estirarse hasta que cada repetición tenga 300 px de ancho, y luego se agregará otra imagen. Luego se comprimirán a 225px.

Ejemplos:

#fondos-background-repeat-repeat {
    background-image: url("../img/background-position-img.png");
    background-repeat: repeat;
    background-size: 50px 50px;
    height: 200px;
    border: 2px solid;
}


#fondos-background-repeat-repeat-x {
    background-image: url("../img/background-position-img.png");
    background-repeat: repeat-x;
    background-size: 50px 50px;
    height: 100px;
    border: 2px solid;
}


#fondos-background-repeat-repeat-y {
    background-image: url("../img/background-position-img.png");
    background-repeat: repeat-y;
    background-size: 20px 20px;
    height: 100px;
    border: 2px solid;
}


#fondos-background-repeat-no-repeat {
    background-image: url("../img/background-position-img.png");
    background-repeat: no-repeat;
    background-size: 50px 50px;
    background-position: center center;
    height: 100px;
    border: 2px solid;
}


#fondos-background-repeat-repeat-space {
    background-image: url("../img/background-position-img.png");
    background-repeat: space;
    background-size: 50px 50px;
    height: 100px;
    border: 2px solid;
}


#fondos-background-repeat-repeat-round {
    background-image: url("../img/background-position-img.png");
    background-repeat: round;

Background-position

Es un propiedad de CSS que define la posición de la imagen de fondo.

Sintaxis

background-position: [porcentaje | palabras clave posición | center];

Admite 3 valores, que son los siguientes:

  • Un porcentaje: La imagen de fondo se colocará al 0 % en el eje horizontal y al 0 % en el eje vertical, lo que significa la esquina superior izquierda del elemento.
  • Palabras clave de posición: Puede utilizar una combinación de palabras clave de posición: center, top, bottom y left right.
  • Center: La imagen de fondo se colocará en el centro del elemento.

Ejemplos:

#fondos-background-position-porcentaje {
    background-image: url("../img/background-position-img.png");
    background-position: 0% 0%;
    background-repeat: no-repeat;
    background-size: 50px 50px;
    height: 100px;
    border: 2px solid;
    background-color: aqua;
}


#fondos-background-position-porcentaje2{
    background-image: url("../img/background-position-img.png");
    background-position: top right;
    background-repeat: no-repeat;
    background-size: 50px 50px;
    height: 100px;
    border: 2px solid;
    background-color: aqua;
}


#fondos-background-position-porcentaje3{
    background-image: url("../img/background-position-img.png");
    background-position: center center;
    background-repeat: no-repeat;
    background-size: 50px 50px;
    height: 100px;
    border: 2px solid;
    background-color: aqua;
}
