TUTORIAL CSS3 - Efectos de Texto y Fuentes

Propiedades para modificar el texto y la fuente de un documento HTML

text-transform

Se usa para establecer la transformación del texto de manera que se muestre en mayúsculas o minúsculas la primera letra de cada palabra o todo el texto por completo.

Sintaxis

text-transform: capitalize|uppercase|lowercase|initial|inherit;

Admite los siguientes valores:

  • capitalize: convierte el primer carácter de cada palabra en mayúsculas.
  • uppercase: convierte el texto en mayúsculas.
  • lowercase: convierte el texto en minúsculas.

Valor por defecto

none

Ejemplos:

text-transform: capitalize;
text-transform: uppercase;
text-transform: lowercase;

Resultado

capitalize

uppercase

LOWERCASE

text-shadow

Se usa para establecer una sombra al texto.

Sintaxis

text-shadow: none|h-shadow v-shadow blur color|initial|inherit;

Admite los siguientes valores:

  • none: no se aplica ninguna sombra.
  • h-shadow: establece la distancia horizontal de la sombra.
  • v-shadow: establece la distancia vertical de la sombra.
  • blur: establece el radio de difuminado de la sombra.
  • color: establece el color de la sombra.

Valor por defecto

none

Ejemplos:

text-shadow: none;
text-shadow: 2px 2px 2px black;
text-shadow: 2px 2px 2px black, 4px 4px 4px #000;

font-variant-ligatures

Se usa para establecer las ligaduras de las fuentes.

Sintaxis

font-variant-ligatures: normal|no-common-ligatures| common-ligatures|historical-lig-value|inherit;

Admite los siguientes valores:

  • normal: no se aplica ninguna ligadura.
  • common-ligatures: se aplican las ligaduras comunes. Que pueden ser "fi", "ffi", "th" o parecidos
  • no-common-ligatures: no se aplica ninguna ligadura común.

Valor por defecto

normal

Ejemplos:

font-variant-ligatures: normal;
font-variant-ligatures: common-ligatures;
font-variant-ligatures: no-common-ligatures;

Resultado

font-family

Se usa para establecer la familia de fuentes del texto.

Sintaxis

font-family: family-name|generic-family|initial|inherit;

Admite los siguientes valores:

  • family-name: establece el nombre de la familia de fuentes.
  • generic-family: establece la familia de fuentes genérica.

En CSS hay 5 familias de fuentes genéricas:

  • serif: tienen un pequeño trazo en los bordes de cada letra. Crean una sensación de formalidad y elegancia.
  • monospace: tienen un ancho fijo. Crean un aspecto mecánico
  • sans-serif: no tienen trazos en los bordes de cada letra. Son más modernas y menos formales.
  • cursive: tienen un trazo que recorre cada letra. Son más informales y creativas.
  • fantasy: tienen un trazo que recorre cada letra. Son más informales y creativas.

Valor por defecto

serif

Ejemplos:

font-family: Arial, Helvetica, sans-serif;
font-family: "Times New Roman", Times, serif;
font-family: "Courier New", Courier, monospace;
font-family: "Lucida Sans Unicode", "Lucida Grande", sans-serif;

font-variant-caps

Se usa para establecer el tipo de capitalización de las fuentes.

Sintaxis

font-variant-caps: normal|small-caps|all-small-caps|petite-caps|all-petite-caps|unicase|titling-caps|inherit;

Admite los siguientes valores:

  • normal: no se aplica ninguna capitalización.
  • small-caps: se aplican las mayúsculas pequeñas.
  • all-small-caps: se aplican las mayúsculas pequeñas a todas las letras.
  • petite-caps: se aplican las mayúsculas pequeñas.
  • all-petite-caps: se aplican las mayúsculas pequeñas a todas las letras.
  • unicase: se aplican las mayúsculas pequeñas a todas las letras.
  • titling-caps: se aplican las mayúsculas pequeñas a todas las letras.

Valor por defecto

Ejemplos:

font-variant-caps: all-small-caps;
font-variant-caps: petite-caps;

font-feature-settings: "unic";
font-family: Arial, Helvetica, sans-serif;
font-variant-caps: unicase;

font-variant-caps: all-petite-caps;

font-feature-settings: "titl";
font-family: 'League Spartan', sans-serif;

text-rendering

Se usa para establecer el modo de renderizado del texto.

Sintaxis

text-rendering: auto|optimizeSpeed|optimizeLegibility|geometricPrecision|initial|inherit;

Admite los siguientes valores:

  • auto: establece el modo de renderizado automático.
  • optimizeSpeed: establece el modo de renderizado para optimizar la velocidad.
  • optimizeLegibility: establece el modo de renderizado para optimizar la legibilidad. En fuentes menores de 20px utiliza las ligaduras de las letras en sílabas como "fi"
  • geometricPrecision: establece el modo de renderizado para optimizar la precisión geométrica.

Valor por defecto

auto

Ejemplos:

text-rendering: auto;
text-rendering: optimizeSpeed;
text-rendering: optimizeLegibility;
text-rendering: geometricPrecision;

text-orientation

Se usa para establecer la orientación del texto. Solo afecta cuando "writing-mode" no está definido a "horizontal-tb"

Sintaxis

text-orientation: mixed|upright|sideways-right|sideways-left|initial|inherit;

Admite los siguientes valores:

  • mixed: establece la orientación del texto en vertical y horizontal.
  • upright: establece la orientación del texto en horizontal.
  • sideways: establece la orientación del texto como lo harían horizontalmente pero girada 90º en sentido de las agujas del relok.

Valor por defecto

mixed

Ejemplos:

text-orientation: mixed;
text-orientation: upright;
text-orientation: sideways;

Google Fonts

Google Fonts es un servicio gratuito que ofrece fuentes de Google para que las utilices en tus sitios web. Puedes ver todas las fuentes disponibles en https://fonts.google.com/

Para utilizar una fuente de Google Fonts, debes incluir el siguiente código en la sección <head> de tu documento HTML:

<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">

En el ejemplo anterior, se incluye la fuente Roboto. Para incluir otra fuente, debes reemplazar Roboto por el nombre de la fuente que deseas utilizar.

También podemos añadir efectos a las fuentes añadiendo la palabra effect seguido del nombre del efecto con la barra vertical "|" podemos añadir varios efectos 

<link href="https://fonts.googleapis.com/css?family=Roboto&effect=shadow-multiple|fire" rel="stylesheet">

Ejemplos

font-family: 'Roboto', sans-serif;

Añadimos las clases que queremos al elemento seleccionado

<p class="font-effect-fire font-effect-shadow-multiple">

font

La propiedad font es una propiedad abreviada para las propiedades font-style, font-variant, font-weight, font-size, line-height y font-family.

La propiedad font se utiliza para establecer todas las propiedades de fuente en una sola declaración.

Sintaxis

font: font-style font-variant font-weight font-size/line-height font-family;

Admite los siguientes valores:

  • font-style: normal, italic, oblique
  • font-variant: normal, small-caps
  • font-weight: normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900
  • font-size: xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, 10px, 12pt, 80%, 150%
  • line-height: normal, number, length, %
  • font-family: nombre de la fuente, serif, sans-serif, cursive, fantasy, monospace

Valor por defecto

font: normal normal normal medium/normal serif;

Ejemplos

font: italic small-caps bold 12px/30px Georgia, serif;
