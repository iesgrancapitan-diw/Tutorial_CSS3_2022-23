TUTORIAL CSS3 - Bordes

Propiedades para modificar los bordes de los elementos de un documento HTML

border-style

Indica el estilo del borde. Es importante definir un border-style para que el borde se pueda ver.

Se permiten los siguientes valores para dicha propiedad

  • dotted - Define un borde punteado.
  • dashed - Define un borde discontinuo.
  • solid - Define un borde sólido.
  • double - Define un borde doble.
  • groove - Define un borde ranurado 3D. El efecto depende del valor del color del borde.
  • ridge - Define un borde estriado 3D. El efecto depende del valor del color del borde.
  • inset - Define un borde de inserción 3D. El efecto depende del valor del color del borde.
  • outset - Define un borde inicial 3D. El efecto depende del valor del color del borde.
  • none - No define ningún borde
  • hidden - Define un borde oculto

Sintaxis

border-style: {tipo de estilo | [estilo borde derecho] | [estilo borde inferior] | [estilo borde izquierdo]};

Admite de uno a cuatro valores, que son los siguientes:

  • Un valor: se aplica a los cuatros lados.
  • Dos valores. se aplica el primero a arriba y abajo y el segundo a derecha e izquierda.
  • Tres valores: se aplica el primero a arriba,el segundo a derecha e izquierda y el tercero a abajo.
  • Cuatro valores: se aplica el primero a arriba, el segundo a la derecha, el tercero abajo y el cuarto a la izquierda.

border-width

Indica que el ancho de los bordes. El ancho se puede establecer con un tamaño específico (en px, pt, cm, em, etc.) o usando uno de los tres valores predefinidos: thin, medium, or thick:

Sintaxis

border-width: {ancho borde | [ancho borde derecho] | [ancho borde inferior] | [ancho borde izquierdo]};

Admite de uno a cuatro valores, que son los siguientes:

  • Un valor: se aplica a los cuatros lados.
  • Dos valores. se aplica el primero a arriba y abajo y el segundo a derecha e izquierda.
  • Tres valores: se aplica el primero a arriba,el segundo a derecha e izquierda y el tercero a abajo.
  • Cuatro valores: se aplica el primero a arriba, el segundo a la derecha, el tercero abajo y el cuarto a la izquierda.

Ejemplos:

border-width: 4px;
