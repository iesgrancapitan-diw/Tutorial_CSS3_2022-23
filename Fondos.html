TUTORIAL CSS3 - Fondos

Propiedades para modificar los fondos de los elementos de un documento HTML

background-attachment

Propiedad que establece si una imagen de fondo se desplaza con el resto de la página, es fija o se desplaza con el texto.

Sintaxis

background-attachment: scroll|fixed|local|initial|inherit;
  • scroll: Valor por defecto.La imagen de fondo se desplazará con la página
  • fixed: La imagen de fondo no se desplazará con la página
  • local: La imagen de fondo se desplazará con el contenido del elemento

Ejemplo

 background-attachment: scroll;
SCROLL. El fondo está fijo en relación con el elemento en sí y no se desplaza con su contenido

background-clip

Propiedad que define cuánto debe extenderse el fondo (color o imagen) dentro de un elemento.

Sintaxis

background-clip: border-box|padding-box|content-box|initial|inherit;
  • border-box: Valor por defecto. El fondo se extiende detrás del borde
  • padding-box: El fondo se extiende hasta el borde interior del borde.
  • content-box: El fondo se extiende hasta el borde del cuadro de contenido.

Ejemplos

Background-color

La propiedad background-color establece el color de fondo de un elemento. El fondo de un elemento es el tamaño total del elemento, incluido el relleno y el borde (pero no el margen).

Sintaxis

background-color: color|transparent|initial|inherit;

Puede admitir estos valores:

  • color: Especifica el color de fondo, admite muchos colores y además varias formas de aplicarle estos colores.

      • Hay varias formas de aplicar color con la propiedad background-color:

    • Escribiendo un color.
      • background-color: coral;
    • Con un valor hexadecimal.
      • background-color: #92a8d1;
    • Con un valor RGB.
      • background-color: rgb(201, 76, 76);
    • Con un valor RGBA.
      • background-color: rgba(201, 76, 76, 0.3);
    • Con un valor HSL.
      • background-color: hsl(89, 43%, 51%);
    • Con un valor HSLA.
      • background-color: hsla(89, 43%, 51%, 0.3);
  • transparent: Especifica que el color de fondo debe ser transparente. 
    • background-color: transparent;
  • initial: Establece esta propiedad en su valor predeterminado.
    • background-color: initial;
  • Inherits: Hereda esta propiedad de su elemento padre.
    • background-color: inherit;

background-size

Define el tamaño de la imagen de fondo.

Sintaxis

background-size: auto | length | cover | contain;

Admite un solo valor, además de los globales:

  • auto: Valor por defecto. La imagen de fondo se muestra en su tamaño original. 
    • background-size: auto;

  • lenght: Establece el ancho y alto de la imagen de fondo. El primer valor establece el ancho, el segundo valor establece la altura. Si solo se proporciona un valor, el segundo se establece en "auto". 
    • background-size: 100px 1000px;

  • cover: Cambia el tamaño de la imagen de fondo para cubrir todo el contenedor, incluso si tiene que estirar la imagen o cortar un poco uno de los bordes. 
    • background-size: cover;

  • contain: Cambia el tamaño de la imagen de fondo para asegurarse de que la imagen sea completamente visible 
    • background-size: contain;

Background-image

La propiedad background-image establece una o más imágenes de fondo para un elemento. De forma predeterminada, una imagen de fondo se coloca en la esquina superior izquierda de un elemento y se repite tanto vertical como horizontalmente.

Sintaxis

background-image: url|none|initial|inherit;

Admite de uno a cuatro valores, que son los siguientes:

  • url: La URL de la imagen. Para especificar más de una imagen, separe las URL con una coma.
  • none: No se mostrará ninguna imagen de fondo.
  • initial: Establece esta propiedad en su valor predeterminado.
  • inherit: Hereda esta propiedad de su elemento padre.

Ejemplos:

background-image: url(../img/fondos-paisaje.jpg), url(../img/fondos-tree.png);
height: 500px;
background-image: url(../img/fondo-pizza.png), url(../img/fondos-png.png), url(../img/fondo-comida.png), url(../img/fondo-manzana.png), url(../img/fondo-naranja.png), url(../img/fondo-mantel.jpg);
background-repeat: no-repeat, no-repeat, no-repeat, no-repeat, no-repeat;
background-position: 10% 10%, 90% 70%, 40% 20%, 20% 80%, 75% 12%;
background-size: 150px, 150px, 150px, 90px, 100px, 100%;

Background-color

La propiedad background-color establece el color de fondo de un elemento. El fondo de un elemento es el tamaño total del elemento, incluido el relleno y el borde (pero no el margen). Para más información sobre los degradados visite el apartado de colores y degradados.

Sintaxis

background-color: color|transparent|initial|inherit;

Puede admitir estos valores:

  • color: Especifica el color de fondo, admite muchos colores y además varias formas de aplicarle estos colores.
  • conic-gradient(): Establece un degradado cónico como imagen de fondo. Debes definir al menos dos colores.
  • linear-gradient(): Establece un degradado lineal como imagen de fondo. Debes definir al menos dos colores (de arriba a abajo).
  • radial-gradient(): Establece un degradado radial como imagen de fondo. Debes definir al menos dos colores (del centro a los bordes).
  • repeating-conic-gradient(): Repite un gradiente cónico.
  • repeating-linear-gradient(): Repite un gradiente lineal.
  • repeating-radial-gradient(): Repite un degradado radial.
    • Auto: la imagen se muestra con el tamaño original.
    • Length: la imagen se muestra con el tamaño que tu introduzcas, con sus respectivas unidades en altura y anchura.
    • Cover: la imagen ocupa todo el fondo, sin importarle el tamaño de esta.
    • Contain: le cambia el tamaño a la imagen para que sea totalmente visible, con respecto al contenedor.

    Ejemplos:

    background-size: auto;
    background-size: 100px 100px;
    background-size: contain;
    background-size: cover;

background-repeat

Indica si la imagen se va repetir y como se va a repetir. Un fondo de imagen puede ser repetido sobre el eje horizontal, el eje vertical, ambos ejes, o no estar repetido.

Sintaxis

background-repeat: repeat-x | repeat-y | repeat | space | round | no-repeat;

Admite un solo valor, además de los globales:

  • Repeat-x: la imagen se repite solo horizontalmente.
  • Repeat-y: la imagen se repite solo verticalmente.
  • Repeat: la imagen se repite tanto horizontal como verticalmente, sin importarle el tamaño de esta.
  • Space: la imagen se repite con cierta separación entre una y otra.
  • Round: la imagen se repite horizontal y verticalmente encajandola en el espacio, sin huecos.
  • No-repeat: no se repite la imagen, por lo que solo se muestra a esta.

Ejemplos:

background-image: url('../img/fondos-logoGC.jpg');
background-size: 150px;
background-repeat: repeat-x;
background-image: url('../img/fondos-logoGC.jpg');
background-size: 150px;
background-repeat: round;
background-image: url('../img/fondos-logoGC.jpg');
background-size: 150px;
background-repeat: repeat-y;

background-position

Define donde se va a colocar inicialmente la imagen de fondo.

  • valor inicial: 0% 0%
  • Se aplica a: todos los elementos
  • Herencia: no
  • Porcentajes: se refiere al tamaño de la caja misma

Sintaxis

background-position: % | cm | left | center | right | top | bottom | xpos ypos;

Admite dos valores que se pueden combinar entre ellos, algunos son:

  • Top, top center y center top: coloca la imagen en el punto medio del eje x.
  • Left, left center y center left: coloca la imagen en el punto medio del eje y.
  • Right, right center y center right: situa la imagen en el extremo derecho centrada.
  • Bottom, bottom center y center bottom: posiciona la imagen en el extremo inferior centrada.
  • Cm: coloca la imagen a los cm introducidos.
  • %: se encarga de colocar la imagen depende del porcentaje de la x e y.
  • xpos ypos: coloca la foto con respecto a unas coordenadas x e y.

Ejemplos:

background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-repeat: no-repeat;
background-position: center;
background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-repeat: no-repeat;
background-position: 5cm;
background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-repeat: no-repeat;
background-position: 75% 100%;

background-origin

La propiedad background-origin especifica el área de origen de un fondo o imagen en determinada caja. para que la propiedad background-position calcule la posición de inicio de un fondo o imagen definida por la propiedad background-image.

  • Valor inicial: padding-box.
  • Aplica a: Todos los elementos.
  • Heredado: No.
  • Porcentajes n/a.
  • Media: Visual.
  • Valor computado: Mismo que valor especificado.
  • Animación: No

    • Sintaxis

    background-origin: padding-box | border-box | content-box;

    Admite de uno a cuatro valores, que son los siguientes:

    • padding-box: El fondo se extiende al borde exterior del padding, la posición es relativa al padding de la caja.
    • border-box: El fondo se extiende al borde exterior (por debajo del borde en el orden z), la posición es relativa al borde de la caja.
    • content-box El fondo se extiende dentro del (recortado al) contenido de la caja, la posición es relativa al contenido de la caja.

    Nota: Si la propiedad background-attachment es igual a "fixed", background-origin no tiene efecto.

    Ejemplos:

    background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-origin: border-box;
background-repeat: no-repeat;
    background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-origin: padding-box;
background-repeat: no-repeat;
    background-image: url('../img/fondos-logo.jpg');
background-size: 150px;
background-origin: content-box;
background-repeat: no-repeat;
background-position: 75% 50%;

    Background-blen-mode

    La propiedad background-blend-mode define el modo de fusión de cada capa de fondo (color y / o imagen).

    Sintaxis

    background-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity;

    Admite varios valores, que son los siguientes:

    • Normal: Esto es el modo predeterminado. Establece el modo de fusión en normal. 
      • background-blend-mode: normal;
    • Multiply: Establece el modo de fusión para multiplicar.
      • background-blend-mode: Multiply;
    • Screen: Establece el modo de fusión en pantalla.
      • background-blend-mode: Screen;
    • Overlay: Establece el modo de fusión en superposición.
      • background-blend-mode: Overlay;
    • Darken: Establece el modo de fusión para oscurecer.
      • background-blend-mode: Darken;
    • Lighten: Establece el modo de fusión para aclarar.
      • background-blend-mode: Lighten;
    • Color-dodge: Establece el modo de fusión para esquivar el color.
      • background-blend-mode: Color-dodge;
    • Saturation: Establece el modo de fusión en saturación.
      • background-blend-mode: Saturation;
    • Color: Establece el modo de fusión en color.
      • background-blend-mode: Color;
    • Luminosity: Establece el modo de fusión en luminosidad.
      • background-blend-mode: Luminosity;

    Background

    La propiedad background es un atajo (shorthand) para definir los valores individuales del fondo en una única regla CSS. Se puede usar background para definir los valores de una o de todas las propiedades siguientes: background-attachment, color, image, position, repeat.

    Sintaxis

    background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;

    La propiedad background se especifica como una o más capas de fondo, separadas por comas.

    La sintaxis de cada capa es la siguiente: Cada capa puede incluir cero o una ocurrencia de cualquiera de los siguientes valores:
    • attachment
    • bg-image
    • position
    • bg-size
    • repeat-style
    El valor bg-size sólo puede ser incluido inmediatamente después de la position, separado con el carácter '/', así: "center/80%". El valor box puede ser incluido cero, una o dos veces. Si se incluye una vez, establece ambos background-origin y background-clip. Si se incluye dos veces, la primera ocurrencia establece background-origin, y el segundo conjunto background-clip. El valor background-color sólo puede ser incluido en la última capa especificada.

    Ejemplo 1

    background: url("../img/fondos-estrellas.jpeg") 10% 85%/350px no-repeat fixed, url("../img/fondos-sol.jpg") 90% 85%/250px no-repeat fixed;

    Ejemplo 2

    Usando background-color

    background: green;

    Ejemplo 3

    Usando bg-image y repeat-style

    background: url("../img/fondos-estrellas.jpeg") repeat-y;

    Ejemplo 4

    Usando box and background-color

    background: border-box red;

    Ejemplo 5

    Una sola imagen, centrada y escalada

    background: no-repeat center/80% url("../img/fondos-estrellas.jpeg");
