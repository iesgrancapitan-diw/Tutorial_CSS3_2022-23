Alineación del texto

text-align

Se usa para establecer la alineación horizontal del texto

Sintaxis

text-align: left|right|center|justify|initial|inherit;

Admite cuatro valores, que son los siguientes:

  • left: alinea el texto a la izquierda.
  • right: alinea el texto a la derecha.
  • center: centra el texto
  • justify: justifica el contenido, es decir, iguala la longitud de todas las líneas del texto

Valor por defecto

left si el texto se escribe de izquierda a derecha (valor ltr en la propiedad direction) y right si el texto se escribe de derecha a izquierda (valor rtl en la propiedad direction)

Ejemplos:

text-align: center;
text-align: left;
text-align: right;
text-align: justify;

text-align-last

Se usa para establecer la alineación horizontal del texto de la última línea de un párrafo

unicode-bidi

Se usa para establecer el nivel de bidi (bidireccionalidad) de un elemento. El nivel de bidi es un número que indica la dirección del texto en un elemento. Los números pares indican que el texto se escribe de izquierda a derecha, mientras que los números impares indican que el texto se escribe de derecha a izquierda.

Sintaxis

unicode-bidi: normal|embed|bidi-override|initial|inherit;

Admite los siguientes valores:

  • normal: el texto se escribe de izquierda a derecha.
  • embed: el texto se escribe de derecha a izquierda.
  • bidi-override: el texto se escribe de derecha a izquierda.

Valor por defecto

normal

Ejemplos:

unicode-bidi: normal;
unicode-bidi: embed;
unicode-bidi: bidi-override;

vertical-align

Se usa para establecer la alineación vertical de un elemento. La alineación vertical se puede establecer en relación con el elemento padre, el elemento anterior o el elemento siguiente.

Sintaxis

vertical-align: baseline|sub|super|top|text-top|middle|bottom|text-bottom|length|percentage|initial|inherit;

Admite los siguientes valores:

  • baseline: alinea el elemento con la línea base del elemento padre.
  • sub: alinea el elemento con la parte inferior del elemento padre.
  • super: alinea el elemento con la parte superior del elemento padre.
  • top: alinea el elemento con la parte superior del elemento padre.
  • text-top: alinea el elemento con la parte superior del texto del elemento padre.
  • middle: alinea el elemento con la parte central del elemento padre.
  • bottom: alinea el elemento con la parte inferior del elemento padre.
  • text-bottom: alinea el elemento con la parte inferior del texto del elemento padre.
  • length: alinea el elemento con la parte superior del elemento padre, pero se desplaza la cantidad especificada.
  • percentage: alinea el elemento con la parte superior del elemento padre, pero se desplaza la cantidad especificada.

Valor por defecto

baseline

Ejemplos:

vertical-align: baseline;
vertical-align: sub;
vertical-align: super;
vertical-align: top;
vertical-align: text-top;
vertical-align: middle;
vertical-align: bottom;
vertical-align: text-bottom;
vertical-align: 50px;
vertical-align: 70%;

Decoración del texto

text-decoration-line

Se usa para establecer la decoración del texto. La decoración del texto se puede establecer en relación con el elemento padre, el elemento anterior o el elemento siguiente.

Sintaxis

text-decoration-line: none|underline|overline|line-through|initial|inherit;

Admite los siguientes valores:

  • none: no se aplica ninguna decoración.
  • underline: se aplica una línea debajo del texto.
  • overline: se aplica una línea encima del texto.
  • line-through: se aplica una línea a través del texto.

Valor por defecto

none

Ambos valores

También podemos usar varios valores a la vez como por ejemplo underline y overline

Ejemplos:

text-decoration-line: none;
text-decoration-line: underline;
text-decoration-line: overline;
text-decoration-line: line-through;
text-decoration-line: underline overline;

Resultado

Underline

Overline

Line-through

Fuentes

font-family

Se usa para establecer la familia de fuentes del texto.

Sintaxis

font-family: family-name|generic-family|initial|inherit;

Admite los siguientes valores:

  • family-name: establece el nombre de la familia de fuentes.
  • generic-family: establece la familia de fuentes genérica.

En CSS hay 5 familias de fuentes genéricas:

  • serif: tienen un pequeño trazo en los bordes de cada letra. Crean una sensación de formalidad y elegancia.
  • monospace: tienen un ancho fijo. Crean un aspecto mecánico
  • sans-serif: no tienen trazos en los bordes de cada letra. Son más modernas y menos formales.
  • cursive: tienen un trazo que recorre cada letra. Son más informales y creativas.
  • fantasy: tienen un trazo que recorre cada letra. Son más informales y creativas.

Valor por defecto

serif

Ejemplos:

font-family: Arial, Helvetica, sans-serif;
font-family: "Times New Roman", Times, serif;
font-family: "Courier New", Courier, monospace;
font-family: "Lucida Sans Unicode", "Lucida Grande", sans-serif;

font-style

Se usa para establecer el estilo de la fuente.

Sintaxis

font-style: normal|italic|oblique|initial|inherit;

Admite los siguientes valores:

  • normal: establece el estilo normal de la fuente.
  • italic: establece el estilo cursiva de la fuente.
  • oblique: establece el estilo oblicuo de la fuente. (Parecido a italic pero menos soportado)

Valor por defecto

normal

Ejemplos:

font-style: normal;
font-style: italic;
font-style: oblique;

Se usa para establecer el tamaño de la fuente.

  • x-small: establece el tamaño de la fuente más pequeño.

  • larger: establece el tamaño de la fuente más grande.
  • length: establece el tamaño de la fuente en unidades de longitud.
  • percentage: establece el tamaño de la fuente en porcentaje.

medium

Google Fonts

Google Fonts es un servicio gratuito que ofrece fuentes de Google para que las utilices en tus sitios web. Puedes ver todas las fuentes disponibles en https://fonts.google.com/

Para utilizar una fuente de Google Fonts, debes incluir el siguiente código en la sección <head> de tu documento HTML:

<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">

En el ejemplo anterior, se incluye la fuente Roboto. Para incluir otra fuente, debes reemplazar Roboto por el nombre de la fuente que deseas utilizar.

También podemos añadir efectos a las fuentes añadiendo la palabra effect seguido del nombre del efecto con la barra vertical "|" podemos añadir varios efectos 

<link href="https://fonts.googleapis.com/css?family=Roboto&effect=shadow-multiple|fire" rel="stylesheet">

Ejemplos

font-family: 'Roboto', sans-serif;

Añadimos las clases que queremos al elemento seleccionado

<p class="font-effect-fire font-effect-shadow-multiple">

font

La propiedad font es una propiedad abreviada para las propiedades font-style, font-variant, font-weight, font-size, line-height y font-family.

La propiedad font se utiliza para establecer todas las propiedades de fuente en una sola declaración.

Sintaxis

font: font-style font-variant font-weight font-size/line-height font-family;

Admite los siguientes valores:

  • font-style: normal, italic, oblique
  • font-variant: normal, small-caps
  • font-weight: normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900
  • font-size: xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, 10px, 12pt, 80%, 150%
  • line-height: normal, number, length, %
  • font-family: nombre de la fuente, serif, sans-serif, cursive, fantasy, monospace

Valor por defecto

font: normal normal normal medium/normal serif;

Ejemplos

font: italic small-caps bold 12px/30px Georgia, serif;
