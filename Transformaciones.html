TUTORIAL CSS3 - Transformaciones

Propiedades para realizar todo tipo de efectos visuales, incluido 2D y 3D

transform: rotate

Es una funcion a la cual le podemos indicar los grados de inclinación

Sintaxis

transform: rotate(grados);

Admite de un valor, que es el siguiente:

  • Un valor: Un ángulo en el que se rota el elemento. En caso positivo, rotará en sentido horario, si es negativo, en sentido antihorario.

Ejemplos:

background-color: darkviolet;
width: 100px;
height: 100px;
transform: rotate(120deg);

transform: rotate3d

Es una funcion CSS la cual rota un elemento alrededor un eje fijo, el cual al ser en 3D tendrá 3 puntos en las coordenadas, describiendo así el centro de rotación.

Sintaxis

transform: rotate3d(x, y, z, grados);

Admite hasta cuatro valores, que son los siguientes:

  • Primer valor: Define la coordenada X para el centro de rotación.
  • Segundo valor: Define la coordenada Y para el centro de rotación.
  • Tercer valor: Define la coordenada Z para el centro de rotación.
  • Cuarto valor: Un ángulo en el que se rota el elemento. En caso positivo, rotará en sentido horario, si es negativo, en sentido antihorario.

Ejemplos:

background-color: darkviolet;
width: 100px;
height: 100px;
transform: rotate3d(3, 1, -1, 120deg);
SIN ROTAR
TRANSFORMACIÓN

transform: translate

Permite mover el elemento u objeto a lo largo del eje de coordenadas en un plano 2D,es decir, a lo largo del eje X e Y .

Sintaxis

transform: translate(x, y);

Admite dos valores, que son los siguientes:

  • 1º Valor: valor que representa el translado a lo largo del eje de absisas, es decir, a lo largo del Eje X.
  • 2º Valor: valor que representa el translado a lo largo del eje de ordenadas, es decir, a lo largo del Eje Y

Cuando hablamos del valor "y", si no se especifíca, su valor es 0.

Ejemplos:

background-color: darkviolet;
height: 100px;
width: 100px;
transform: translate(120, 40);
Aquí el resultado del ejemplo del código fuente puesto arriba, es crear un cuadrado transladado 120px a lo largo del Eje X y 40px a lo largo del eje Y.

transform: translate3D

Dentro del CSS hay una función llamada translate3d() donde mueve el elemento en un espacio 3D, por lo que lo movera en un espacio en el que hay 3 vectores( vectores: "x", "y" y "z")

Sintaxis

transform: translate3d(x, y, z);

Admite tres valores (debido a los vectores que representa.):

  • 1º valor: representa el vector del eje de absisa (Eje X) valorado en px
  • 2º valor: representa el vector del eje de ordenadas (Eje Y) valorado en px
  • 3º valor: representa el vector del eje cota (Eje Z) valorado en px

Ejemplos:

background-color: darkviolet;
height: 100px;
width: 100px;
transform: perspective(500 px) translate3d(110px, 50px, 180px);
Aqui el resultado del ejemplo del código fuente puesto arriba, es crear un cuadrado y moverlo en un espacio 3D, movido por 3 vectores, necesitas hacer un perspective que se ha explicado con anterioridad.

transform: skew

Función que provoca una deformación en un plano 2D al objeto, a lo largo del Eje X y del Eje Y

Sintaxis

transform: skew(x, y);

Admite dos valores, ambos medidios en grados(deg), que son los siguientes:

  • 1º Valor: Ángulo que tomaríamos para deformar el objeto a lo largo del eje de absisa, es decir, a lo largo del Eje X

Ejemplos:

background-color: darkviolet;
width: 100px;
height: 100px;
transform: skew(20deg, 10deg);

transform: matrix

La función matrix combina todos los métodos de transformación 2D en uno, tomando 6 parámetros.

Sintaxis

transform: matrix(scaleX(),skewY(), skewX(), scaleY(),translateX(), translateY());

Admite 6 parámetros que serían todos los dichos anteriormente (No tienes que poner la función de cada uno, solo poner los valores de ellos):

  • Los 4 primero valores: Describen la transformación linear
  • Los 2 último valores: Describen el movimiento del a lo largo del eje

En caso de no poner ningun valor en algun caso no toma valores por defecto, por lo que habría que poner siempre valores, en caso de no querer un valor asignarle el 0.

Ejemplos:

background-color: darkviolet;
height: 100px;
width: 100px;
transform: matrix(2, 0.5, -0.5, 1, 180, 0);
En el ejemplo del código fuente de arriba tomamos 6 parámetros para poder hacer una transformación en 2D con el fin de hacer en la figura/elemento lo que deseemos dandole siempre que no quisieramos la utilidad de una función el valor 0, porque al no ponerle un valor a una función todo cambiaría.

transform: matrix3d

La función matrix3d combina todos los métodos de transformación 3D y el uso de la función perspective() en uno, tomando 16 parámetros.

Sintaxis

transform: matrix3d(a1, b1, c1, d1, a2, b2, c2, d2, a3, b3, c3, d3, a4, b4, c4, d4);

Admite 16 parámetros que serían todos los dichos anteriormente (Todos los parámetros serían recogidos por las funciones 3d --> scale3d(), rotate3d(), rotateX(), rotateY(), rotateZ(), scaleZ(), translate3d(), translateZ(), perspective()):

  • Valores para la transformación lineal a aplicar al elemento u objeto: a1, b1, c1, d1, a2, b2, c2, d2, a3, b3, c3, d3, d4
  • Valores para la traducción a aplicar al elemento u objeto: a4, b4, c4

Cuando creamos una transformación en 3D, podemos usar una de las otras funciones de transformación 3D, o usar matrix3d con todos los argumentos anteriormente citados.

Ejemplos:

background-color: darkviolet;
width: 100px;
height: 100px;
transform: matrix3d(2,-1,-5,0,0,1.5,7,0,2,0,4,0,170,20,1.4,1.1);
En el ejemplo del código fuente de arriba tomamos 16 parámetros para poder hacer una transformación en 3D con el fin de crear en la figura/elemento lo que deseemos, en caso de no querer usar un valor tendrímos que poner como valor por defecto un 0, porque al no ponerle un valor a una función todo cambiaría.

transform: origin

Esta transformación permite modificar el origen de las transformaciones de un elemento

Sintaxis

transform-origin: (y-offset, x-offset, z-offset);

Admite 3 parámetros: que cada uno representa un eje:

  • 1º y-offset: indica la distancia del borde superior de la caja se establece el origen de la transformación se mide en length y porcentage, también se puede valorizar con palabras claves como top, bottom o center pero se usaría de diferente forma --> y-offset-keyword
  • 2º x-offset: indica la distancia al borde izquierdo de la caja se establece el origen de la transformación se mide en length y porcentage, también se puede valorizar con palabras claves como left, right o center pero se usaría de difernete forma --> x-offset-keyword
  • 3º z-offset: indica que tan lejos del ojo del usuario se establece el origen de posision z = 0, se mide en lenght y nunca en porcentage (es una declaración no válida).
  • 4º offset-keyword: usa palabras clave como left, right, top bottom o center que describen el correspondiente desplazamiento

Ejemplos:

background-color: darkviolet;
width: 100px;
height: 100px;
transform: translateX(120px) skewX(30deg);
transform-origin: 100% -30%;

ORIGIN

transform-box

La propiedad CSS transform-box define el cuadro de diseño con el que se relacionan las propiedades transform y transform-origin.

Sintaxis

transform-box: (Propiedad de transform-box);

La propiedad transform-box se especifica con una de las palabras claves que se enumeran a continuación:

  • content-box: el content-box se utiliza como cuadro de referencia. Por ejemplo: El cuadro de referencia de una tabla es el border box de la caja que lo contiene, no de la caja de la tabla.
  • border-box: el border-box se utiliza como cuadro de referencia. Por ejemplo: El cuadro de referencia de una tabla es el border box de la caja que lo contiene, no de la caja de la tabla.
  • fill-box: el cuadro delimitador del objeto se utiliza como cuadro de referencia.
  • stroke-box: el cuadro delimitador del trazo se utiliza como cuadro de referencia.
  • view-box: la ventana gráfica SVG más cercana se utiliza como cuadro de referencia. Si se especifica un atributo viewBox para el elemento de creación de la ventana gráfica SVG, el cuadro de referencia se coloca en el origen del sistema de coordenadas establecido por el atributo viewBox, y la dimensión del cuadro de referencia se establece en los valores de ancho y alto del atributo viewBox.

Ejemplos:

<svg id="svg" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 50 50">	
    <g><rect id="box" x="10" y="10" width="10" height="10" rx="1" ry="1" stroke="black" fill="none" /></g>	
</svg>
#svg{
    width:30vh;
    margin: auto;
}

#box {
    transform-origin:50% 50%;
    transform-box: fill-box;
    animation: rotateBox 3s linear infinite;
}

@keyframes rotateBox {
    to {
        transform: rotate(360deg);
}
