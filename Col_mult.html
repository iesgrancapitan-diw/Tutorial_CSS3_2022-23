TUTORIAL CSS3 - Columnas Múltiples

Propiedades para dar un formato de columnas múltiples a los elementos de un documento HTML

column-count

Establece el número de columnas en las que se debe dividir un elemento.

Sintaxis

column-count: número|auto|initial|inherit;

Admite uno de estos cuatro valores:

  • número: El número de columnas en las que fluirá el contenido del elemento.
  • initial: (Valor predeterminado) El número de columnas vendrá determinado por otras propiedades, como por ejemplo, column-width.
  • initial: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

Ejemplos:

column-count: 3;

column-gap

Establece la longitud del espacio horizontal entre columnas.

Sintaxis

column-gap: longitud|normal|initial|inherit;

Admite uno de los siguientes valores:

  • longitud: valor formado por un número y una unidad, tanto absoluta como relativa.
  • normal: (Valor predeterminado) W3C sugiere 1em.
  • initial: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

Ejemplos:

column-count: 4;
column-gap: 5em;

column-fill

Establece la forma de llenar las columnas, de forma equilibrada o no. El contenido se puede equilibrar o completar de forma secuencial. Sugerencia: si agrega una altura a un elemento de varias columnas, puede controlar cómo el contenido llena las columnas.

Sintaxis

column-fill: balance|auto|initial|inherit;

Admite uno de los siguientes valores:

  • balance: (Valor predeterminado) Llena cada columna con aproximadamente la misma cantidad de contenido, pero no permitirá que las columnas sean más altas que la altura (por lo tanto, las columnas pueden ser más cortas que la altura ya que el navegador distribuye el contenido de manera uniforme horizontalmente).
  • initial: Llena cada columna hasta que alcance la altura, y lo hace hasta que se quede sin contenido (por lo tanto, este valor no necesariamente llenará todas las columnas ni las llenará de manera uniforme).
  • auto: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

Ejemplos:

column-count: 3;
column-fill: balance;
height: 200px;

Sintaxis

column-width: auto|longitud|initial|inherit;

  • longitud: valor formado por un número y una unidad, tanto absoluta como relativa excepto porcentajes.
  • auto: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

Ejemplos:

column-span

Establece el número de columnas que debe abarcar un elemento.

Sintaxis

column-span: none|all|initial|inherit;

Admite uno de los siguientes valores:

  • none: (Valor predeterminado) El elemento se extiende a lo largo de una columna.
  • all: El elemento se extiende, abarcando todas las columnas en una sola.
  • initial: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

column-count: 3;
column-span: none;

column-rule-style

Establece el estilo de la línea de separación entre las columnas.

Sintaxis

column-rule-style: none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit;

Admite uno de los siguientes valores:

  • none: (Valor predeterminado) No existe línea de separación.
  • hidden: la línea de separación se establece como oculta.
  • dotted: la línea de separación se establece como punteada.
  • dashed: la línea de separación se establece como discontinua.
  • solid: la línea de separación se establece como continua.
  • double: la línea de separación se establece como doble.
  • grove: la línea de separación se establece como hundida en 3d.
  • ridge: la línea de separación se establece como estriada en 3d.
  • inset: la línea de separación se establece como hundida provocando que el elemento que encierra parezca estar por debajo del nivel de la superficie de la pantalla.
  • outset: la línea de separación se establece como saliente provocando que el elemento que encierra parezca estar por encima del nivel de la superficie de la pantalla.
  • initial: Establece el valor predeterminado para esta propiedad.
  • inherit: Hereda el valor de esta propiedad del elemento padre.

Ejemplos:

column-count: 3;
column-gap: 40px;
column-rule-style: dotted;

column-rule

Es la union de las propiedades column-rule-width, column-rule-color y column-rule-style. Indica anchura, color y estilo de la línea separadora entre las columnas.

Sintaxis

column-rule: column-rule-width || column-rule-style || column-rule-color;

Admite los valores de las propiedades que engloba en cualquier cantidad y orden, ya que cada uno se especifica de manera diferente. Las propiedades no incluidas serán aplicadas con su valor predeterminado

Ejemplos:

column-count: 3;
column-gap: 40px;
column-rule: 4px double red;
