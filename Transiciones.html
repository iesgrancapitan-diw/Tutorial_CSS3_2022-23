TUTORIAL CSS3 - Transiciones

Propiedades para aplicar transiciones a nuestro archivo

transition-duration

Establece el tiempo que debe tardar una transición en completarse.

Por defecto, el tiempo de transición que utiliza es de 0 segundos, lo que significa que no habrá transición si no le ponemos nosotros una duración.

Sintaxis

transition-duration: [Aquí va la duración que le queramos dar a la transición];

Ejemplos:

div {
width: 100px;
height: 100px;
background: red;
transition-property: width;
transition-duration: 5s;
}
            
div:hover {
width: 200px;
}
Resultado del código de arriba:

Lo que hará será que al pasar el ratón por encima del cuadrado, tardará 5 segundos en aumentar de tamaño.

transition-delay

Especifica la cantidad de tiempo a esperar entre un cambio pedido hacia una propiedad y el comienzo de un efecto de transicion.

Es decir, si ponemos un delay de 5 segundos, la transición comenzará a los 5 segundos.

  • Si no se especifica, el valor por defecto es 0.

  • Si se especifica un valor negativo, el efecto de transición se iniciará inmediatamente.

Sintaxis

transition-delay: [Aqui va la cantidad de tiempo que queramos darle];

Ejemplos:

div {
width: 100px;
height: 100px;
background: red;
transition-property: width;
transition-duration: 5s;
transition-delay: 2s;
}
              
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, el transition-delay, lo que hará es que al poner el cursos en el cuadrado, tardará 2 segundos en comenzar la transición, y luego de esos 2 segundos, tardará otros 5 segundos en completarse.

transition-property

Se usa para definir los nombres de las propiedades CSS en las que el efecto de la transición debe aplicarse.

  • Si no se especifica, el valor por defecto es all, lo que significa que la transición se aplicará a todas las propiedades.

Sintaxis

transition-property: [Aquí definiremos las propiedades a la que se deberá aplicar];

Ejemplos:

div {
width: 100px;
height: 100px;
background: red;
transition-property: width;
transition-duration: 2s;
}
              
div:hover {
width: 300px;
}
Resultas del codigo de arriba:

Con esa propiedad, lo que hará será que al pasar el ratón por encima del cuadrado, se aplicará la transición a la propiedad width, que es la que hemos definido.

transition-timing-function

Es utilizada para especificar el ritmo en el que se producen los cambios durante la transición.

Tipos de transition-timing-function:

  • ease: La transición comienza lenta, acelera en el medio y termina lenta.
  • ease-in: La transición comienza lenta y acelera hasta el final.
  • ease-out: La transición comienza rápida y termina lenta.
  • ease-in-out:La transición comienza lenta, acelera en el medio y termina lenta.
  • linear: La transición tiene la misma velocidad desde el principio hasta el final.

Sintaxis

transition-timing-function: [Aquí especificaremos el ritmo de los cambios];


Ejemplo transition-timing-function: ease:

div {
width: 100px;
height: 50px;
background: red;
color: white;
font-weight: bold;
transition: width 2s;
transition-timing-function: ease;
}
            
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, al poner el cursor encima del cuadrado, especificará un efecto de transición con un comienzo lento, luego rápido y luego un final lento.




Ejemplo transition-timing-function-ease-in:

div {
width: 100px;
height: 50px;
background: red;
color: white;
font-weight: bold;
transition: width 2s;
transition-timing-function: ease-in;
}
                                
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, al poner el cursor encima del cuadrado, especificará un efecto de transición con un comienzo lento.




Ejemplo transition-timing-function: ease-out:

div {
width: 100px;
height: 50px;
background: red;
color: white;
font-weight: bold;
transition: width 2s;
transition-timing-function: ease-out;
}
                                
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, al poner el cursor encima del cuadrado, especificará un efecto de transición con un final lento.




Ejemplo transition-timing-function: ease-in-out:

div {
width: 100px;
height: 50px;
background: red;
color: white;
font-weight: bold;
transition: width 2s;
transition-timing-function: ease-in-out;
}
                                
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, al poner el cursor encima del cuadrado, especificará un efecto de transición con un comienzo y un final lentos.




Ejemplo transition-timing-function: linear:

div {
width: 100px;
height: 50px;
background: red;
color: white;
font-weight: bold;
transition: width 2s;
transition-timing-function: linear;
}
                                
div:hover {
width: 300px;
}
Resultado del codigo de arriba:

En este ejemplo, al poner el cursor encima del cuadrado, especificará un efecto de transición con la misma velocidad de principio a fin.

transition(shorthand)

La propiedad "transition" es una propiedad abreviada para :

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

Sintaxis

transition: [Aquí definiremos las propiedades a la que se deberá aplicar];

Ejemplos:

div {
  width: 100px;
  height: 100px;
  background: red;
  transition: width 2s linear 1s;
}

div:hover {
  width: 300px;
}
Resultas del codigo de arriba:

Con esa propiedad, lo que hará será que al pasar el ratón por encima del cuadrado, se aplicará una transición de 2 segundos, con una velocidad lineal y con un retraso de 1 segundo.
